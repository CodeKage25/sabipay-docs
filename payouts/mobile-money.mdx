## Payout provider codes

When withdrawing via mobile money, pass the `momoProvider` and `momoCountry` fields with the values below.

<Warning>
Payout provider codes are **different** from collection provider codes. Always use the values in this table — not the ones from the payment providers page — when submitting a withdrawal.
</Warning>

### Senegal (XOF)

| `momoProvider` | Provider | `momoCountry` |
|----------------|----------|---------------|
| `wave` | Wave | `senegal` |
| `orange` | Orange Money | `senegal` |
| `free` | Free Money | `senegal` |
| `expresso` | Expresso | `senegal` |

## Example

Withdrawing 5,000 XOF via Wave in Senegal:

```json
{
  "amount": "5000",
  "currency": "XOF",
  "method": "momo",
  "phone": "+221701234567",
  "momoProvider": "wave",
  "momoCountry": "senegal"
}
```
