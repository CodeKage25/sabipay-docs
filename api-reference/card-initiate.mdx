## POST /pay/merchant-api/card

Initiates and charges a card payment in USD.

### Headers

```
x-auth-merchantid: YOUR_MERCHANT_ID
x-auth-apikey: YOUR_API_KEY
Content-Type: application/json
```

### Request body

```json
{
  "reference": "a1b2c3d4-e5f6-4a7b-8c9d-0e1f2a3b4c5d",
  "amount": "10",
  "currency": "USD",
  "pan": "4111111111111111",
  "cvv": "123",
  "expiryMonth": "12",
  "expiryYear": "27",
  "callbackUrl": "https://yoursite.com/callback",
  "customerEmail": "customer@example.com",
  "firstName": "John",
  "lastName": "Doe",
  "phone": "2348100000000",
  "narration": "Order #1234"
}
```

### Response — immediate success

```json
{
  "success": true,
  "message": "Card payment processed",
  "data": {
    "reference": "a1b2c3d4-...",
    "status": "Successful"
  }
}
```

### Response — 3DS redirect required

```json
{
  "success": true,
  "message": "3DS authentication required",
  "data": {
    "requiresRedirect": true,
    "reference": "a1b2c3d4-...",
    "redirect_auth_data": "https://bank.com/3ds/authenticate?..."
  }
}
```

When `requiresRedirect` is `true`, send the customer to the `redirect_auth_data` URL. After authentication they'll return to your `callbackUrl`. Then poll `GET /pay/merchant-api/card/status/:reference` to confirm.
