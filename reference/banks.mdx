## Bank codes (Nigeria)

Use the `bankCode` value when linking a bank account or resolving an account name. These are NIP (NIBSS Instant Payment) codes.

|------|-----------|

| Citibank Nigeria | `000009` |
| Ecobank Nigeria | `000010` |

<Note>
These are 6-digit NIP codes, not the older 3-digit CBN codes. Make sure you're using the correct format — using the wrong code will cause account resolution to fail.
</Note>
